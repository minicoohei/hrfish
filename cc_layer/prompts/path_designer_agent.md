# PathDesignerAgent プロンプトテンプレート

## モデル: sonnet

## Skills 参照
- mirofish-zep: Zepグラフからの背景情報取得
- mirofish-career: イベントタイプ定義・ブロッカールール

## タスク

候補者の情報から、**取りうるキャリアパスを5つ**提案してください。
各パスは方向性が明確に異なり、現実的である必要があります。

## 入力変数

```
graph_id: {{graph_id}}
identity: {{identity_json}}
profiles: {{profiles_json}}
requirement: {{simulation_requirement}}
document_text: {{document_text}}
```

## Zep操作（必須）

パス設計の前にZepグラフを検索し、候補者の背景を把握すること。

```bash
python -m cc_layer.cli.zep_search --graph-id {{graph_id}} --mode entities
python -m cc_layer.cli.zep_search --graph-id {{graph_id}} \
  --query "候補者の強み・弱み・キャリア志向" --mode quick
```

## 前提データ: 組織ピラミッド到達率

**パスの estimated_probability を設定する際に、必ず以下のデータを参照すること。**
組織はピラミッド構造であり、上位ポジションほど到達率は指数関数的に下がる。

| 組織タイプ | 課長相当 | 部長相当 | 執行役員 | 取締役 | CEO/社長 |
|-----------|---------|---------|---------|--------|----------|
| 日系大企業 | 30-50% | 4-10% | 0.5-1% | 0.1% | 0.01% |
| メガバンク | 20-30% | 5% | 0.1% | 0.05% | 0.01% |
| GAFA(L5→) | 60%(L5) | 15%(L6) | 3%(L7) | 1%(Dir) | 0.1%(VP) |
| MBB | 50%(Mgr) | 15%(Prnc) | 2%(Ptr) | -- | -- |
| Big4 | 30%(Mgr) | 8%(Dir) | 1.5%(Ptr) | -- | -- |

Sources: 厚労省「賃金構造基本統計調査」、パーソル総合研究所(2022)、東洋経済/ダイヤモンド人事部長調査、Candor/Levels.fyi、CaseCoach

**estimated_probability の設定ルール:**
- パスの最終ゴールの到達率を上限とする（到達率0.1%のポジションなら、probability > 10%はありえない）
- ただし「そのポジションを目指す候補者」は母集団より優秀なので、到達率の5-20倍程度まで上方修正可
- 例: メガバンク取締役（到達率0.1%）を目指すパス → estimated_probability = 1-2%（優秀な部長候補として20倍補正）

## パス設計ガイドライン

1. **多様性**: 5パスは明確に異なる方向性を持つこと（同じ方向の微修正はNG）
2. **現実性**: 候補者の年齢・スキル・家族状況から到達可能な範囲内
3. **リスクの勾配**: 保守的(1つ)→中庸(2つ)→挑戦的(2つ)のバランス
4. **ブロッカー意識**: mirofish-career の6ブロッカー（住宅ローン、育児、健康等）を考慮
5. **trigger_event**: 各パスが動き出すきっかけとなる出来事を明記
6. **確率の勾配**: 各パスに概算の実現確率を設定する。5パス全部が「頑張ればいける」はNG。**上記ピラミッド到達率を基準にすること**

### 確率を意識したパス設計

パスの最終ゴールが「CPO就任」「IPO」「GAFA入社」など高難度の場合、
**それが実現しなかったときの中間着地点（VP止まり、シリーズA止まり等）** も `fallback_scenario` として明記すること。

```json
{
  "path_id": "path_a",
  "label": "LayerX CPO昇進",
  "direction": "現職で経営層へ",
  "estimated_probability": 0.08,
  "probability_rationale": "CPOは1社1席（到達率0.3%）。候補者のPdM歴9年で20倍補正しても6%。社内競合2名を考慮し8%。",
  "fallback_scenario": "VP of Product止まり（年収1500万、満足度0.8）。CPOにならなくても充分なキャリアだが、経営意思決定からは遠い。",
  ...
}
```

以下のポジションは「1社に1席」であることを常に意識する:
- **CPO / CTO / CFO / COO** — 1社1席。外部から入る難易度は更に高い
- **VP以上** — 1社に1-3席。メガベンチャーでもポスト数は限定的
- **GAFA PM Director以上** — グローバル競争。日本オフィスでは数名
- **メガバンク取締役** — 同期数百人から0.1%。第一選抜に残り続ける必要
- **MBBパートナー** — 同期コホートの1-2%。up-or-out環境
